# Ingame Minesweeper for Minecraft 1.21.11 Fabric

Клиентский Fabric-мод: по клавише **Ж** открывается мини-игра **Сапёр** прямо внутри Minecraft.

## Управление

- **Ж** — открыть Сапёр. На русской раскладке это физическая клавиша `;`, поэтому в настройках Minecraft может отображаться как `;`.
- **ЛКМ по клетке** — открыть клетку.
- **ПКМ по клетке** — поставить/убрать флажок.
- **Esc** — закрыть экран.
- **Новая игра** — перезапуск поля.

## Сборка

Нужны:

- Java 21
- Gradle 8.14+ или 9.x
- Интернет при первой сборке, чтобы Gradle скачал Fabric зависимости

Команда:

```bash
gradle build
```

Готовый `.jar` появится здесь:

```text
build/libs/ingame-minesweeper-1.0.0.jar
```

Положи `.jar` в папку `mods` вместе с Fabric API для Minecraft 1.21.11.

## Версии зависимостей

- Minecraft: 1.21.11
- Fabric Loader: 0.18.1+
- Fabric API: 0.141.4+1.21.11
- Fabric Loom Remap: 1.14-SNAPSHOT
